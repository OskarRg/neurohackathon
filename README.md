### 3. The Neuroscience: Under the Hood
We don't guess if you are stressed. We measure it.

Using the **BrainAccess EEG** headset, we analyze raw neural activity in real-time.
* **The Stress Index:** We compute the ratio between **Beta Waves** (active, anxious thinking) and **Alpha Waves** (relaxed focus). A high ratio triggers the system.
* **Signal Processing & Denoising:** Raw EEG data is noisy. We implemented advanced signal smoothing algorithms (Weighted Rolling Average & Artifact Rejection) to filter out muscle movements (blinking, jaw clenching). We ensure the duck reacts to *your brain*, not your body twitching.
* **Bio-Feedback Loop:** The UI visualizes your state (Zen vs. Panic), giving you immediate feedback to self-regulate.
